Hexo部署

什么是Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用Markdown（或其他标记语言）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

根据Hexo官方文档,需要安装Node.js(且版本不能低于10.13),还需要安装Git方便管理上传至Github的网页。

安装Hexo

可以使用npm安装,终端输入:

npm install -g hexo-cli

或者

npm install hexo
